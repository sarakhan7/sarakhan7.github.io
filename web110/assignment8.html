SQL Commands

  1. SELECT
    SELECT * FROM Customers;
  2. WHERE
    SELECT CustomerName, Address FROM Customers WHERE Country = 'United Kingdom';
  3. COUNT
    SELECT COUNT(*) FROM Orders;
  4. INSERT INTO
    INSERT INTO Shippers (ShipperID, ShipperName, Phone) VALUES (4, 'On Time Delivery', '(503) 555-0123');
  5. UPDATE
    UPDATE Products SET Price = Price + 1;

This lesson really helped demystify SQL for me. I had heard of commands like SELECT and WHERE before, but now I understand how they actually work to pull specific data from a database. Writing queries made me realize how powerful SQL is for organizing and interacting with data. I also learned how INSERT and UPDATE statements can add or change information directly in a table, which feels like real control over a database system. I can definitely see myself using SQL in future development work, especially in full-stack apps where you need to fetch and update live data efficiently.
