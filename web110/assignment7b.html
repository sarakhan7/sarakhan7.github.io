    The Categories table includes: CategoryID, CategoryName, Description
  2. What is the primary key for the Customers table?
    CustomerID
  3. How are the Orders table and the OrderDetails table related?
    They are related through OrderID (primary key in Orders, foreign key in OrderDetails), creating a one-to-many relationship where each order can have multiple details.
